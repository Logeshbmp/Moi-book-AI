# Moi Book AI

AI-powered Moi (Gift Contribution) Management App for Weddings and Family Functions.

## Features

- Voice-based Moi Entry
- Tamil & English Support
- Automatic Amount Detection
- Function-wise Records
- Search by Name
- Moi Return History
- PDF Report Generation

## Example

Voice Input:
"Murugan 1001 rooba kuduthaar"

AI Output:
Name: Murugan
Amount: ₹1001

## Tech Stack

- Flutter
- Firebase
- Gemini AI
- Speech-to-Text
